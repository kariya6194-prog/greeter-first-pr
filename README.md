# Greeter

A tiny practice project for making your **first pull request**.

## What it does

`greet.py` builds a friendly greeting message from a name. That's all it does —
small on purpose, so the focus stays on the workflow, not the code.

## Usage

```bash
python greet.py World
# -> Hello, World!
```

## Project layout

| File                  | Purpose                          |
| --------------------- | -------------------------------- |
| `greet.py`            | The greeting function and CLI    |
| `tests/test_greet.py` | A few basic tests for the module |

## Contributing

Spotted a typo or a rough edge? Open a pull request — even one-line fixes are
welcome. That is exactly how most people make their very first contribution.
